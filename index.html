Acerca de mí

Estoy apasionado por explorar las últimas tecnologías para abordar diversos desafíos tecnológicos, ya sea en Windows, MacOS o Linux. Además, cuento con habilidades sólidas en el desarrollo y manejo de tecnologías clave como CSS para el diseño web, JavaScript para la interactividad, así como MySQL y SQL para la gestión de bases de datos. También tengo experiencia en el desarrollo de aplicaciones utilizando lenguajes como Java, C# y C. Estas habilidades me permiten abordar una amplia gama de proyectos con confianza y eficacia. Estoy preparado para aplicar estas competencias en proyectos desafiantes y colaborar con equipos para alcanzar soluciones innovadoras.

Herramientas

En esta sección, se muestran los programas y lenguajes que he empleado en proyectos de programación como en iniciativas personales. Se proporciona una descripción de cada herramienta.

Diseño

Figma

Plataforma de diseño colaborativo para crear y prototipar interfaces digitales de forma eficiente.

Illustrator

Software de diseño vectorial para crear gráficos, ilustraciones y logotipos de alta calidad con precisión.

Photoshop

Herramienta de edición de imágenes y diseño gráfico para retocar fotos y crear composiciones artísticas.

Bootstrap

Framework de desarrollo web que facilita la creación de sitios modernos y responsivos con componentes preestablecidos.

Front-End

JavaScript

Lenguaje de programación utilizado para agregar interactividad y funcionalidad dinámica a las páginas web.

Cascading Style

Lenguaje de estilo utilizado para controlar el diseño y la presentación de los documentos HTML.

HyperText Markup

Lenguaje de marcado estándar utilizado para crear la estructura y el contenido de las páginas web.

JQuery

Biblioteca de JavaScript que simplifica la manipulación del DOM y la creación de efectos y animaciones en páginas web.

Back-End

Node

Entorno de ejecución de JavaScript del lado del servidor que permite construir aplicaciones web escalables.

C Sharp

Lenguaje de programación utilizado principalmente para crear aplicaciones de escritorio y móviles en .NET.

Java

Lenguaje de programación utilizado por su portabilidad y su amplio uso en el desarrollo de aplicaciones empresariales y móviles.

C++

Lenguaje de programación de alto rendimiento y utilizado en juegos, sistemas operativos y aplicaciones de software.

Bases de datos

SQL

Lenguaje para la gestión de bases de datos relacionales, utilizado para consultar y manipular datos en sistemas RDBMS.

MySQL

Sistema de gestión de bases de datos relacional de código abierto, popular en aplicaciones web y empresariales por su rendimiento.

Mongo DB

Base de datos NoSQL de documentos, ideal para aplicaciones modernas que manejan datos no estructurados.

DBeaver

Herramienta de administración de bases de datos multiplataforma, que soporta varios sistemas de gestión y ofrece funciones SQL y de diseño de esquemas.

Proyectos

En mi sección de Proyecto, te invito a observar mis proyectos donde se resolvieron diversos desafíos. Desde innovaciones sociales, hasta inspiración y soluciones creativas para un video juego.

Pets INN

PETS INN es un software de reserva de hotelería para mascotas que simplifica la gestión de reservas, comunicación con clientes y facturación. Con perfiles detallados de mascotas y un calendario de disponibilidad, ofrece eficiencia operativa y una experiencia mejorada para los dueños de mascotas.

Vortex

Vortex nació con la idea de crear un juego de aventuras en 2D ambientado en un universo de ciencia ficción, donde los jugadores encarnan a un científico errante que se aventura a través de mundos desolados para rescatar civilizaciones devastadas. Enfrentando criaturas mutantes y resolviendo acertijos, el científico lucha por salvar el futuro de la galaxia.
